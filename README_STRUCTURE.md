# 3D_Navigation_project 구조

## 폴더 역할

- `paper_notes/` : 논문 읽기 노트
- `notes/` : 연구 메모, 주간 회고, 아이디어 정리
- `isaac/` : Isaac Sim 관련 설정, 체크리스트, 실행 메모
- `isaaclab/` : Isaac Lab 관련 설치/연결 작업용 폴더
- `scripts/` : Python 테스트 코드, 데이터 추출 코드
- `experiments/` : 실험별 설정/기록
- `logs/` : 실행 로그, 에러 로그, 디버깅 메모
- `results/` : 스크린샷, 플롯, 결과물
- `assets/` : 이미지, PDF, 보조 자료
- `docs/` : 발표자료 초안, 정리 문서

## 추천 사용법

- 논문 정리는 `paper_notes/`
- Isaac Sim 관련 세팅 메모는 `isaac/`
- 직접 실행하는 테스트 스크립트는 `scripts/`
- 실험 단위 기록은 `experiments/`
- 나중에 발표/포트폴리오는 `docs/`와 `results/` 중심으로 정리
